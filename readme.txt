Menampilkan daftar radio streaming Islam tepercaya, merupakan versi widget web resmi dari app android RII

== Description ==

Plugin website untuk platform wordpress ini adalah versi plugin widget resmi dari aplikasi android RII -  Radio Islam Indonesia.

Menampilakn daftar radio streaming Islam tepercaya di Indonesia yang menyandarkan dakwahnya berdasar al-Qur`an, as-Sunnah, seperti yang dipahami oleh para shahabat radhiyallaahu anhum.

= Penting =
Plugin widget ini masih jauh dari sempurna, kami sangat mengharap masukan Anda demi pengembangan aplikasi android RII dan plugin widget RII ini  melalui email: `oasemedia.dev@gmail.com`

= Fitur =
* Lebih dari 40 saluran radio
* 2 Skin menarik
* Desain responsif
* Kompatibel untuk semua tema
* Status realtime dari tiap radio dengan refresh rate yang bisa Anda tentukan sendiri.
* Sistem urutan daftar radio dinamis, berdasar status LIVE, online, dan offline sesuai dengan jumlah pendengar.

== Installation ==

= Menggunakan built-in plugin installer =

1. Pergi ke Plugins > Add New.
1. Cari dengan keyword "Radio Islam".
1. Temukan "Radio Islam" plugin untuk diinstall.
1. Klik "install now" untuk menginstall plugin.
1. Muncul window untuk mengkonfirmasi apakah akan melanjutkan proses install plugin.
1. Klik ok untuk menginstall.
1. Jika sukses, klik "Activate" untuk mengaktifkan plugin.

= Menggunakan cara manual =

1. Download "Radio Islam" plugin.
1. Ekstrak file ke dalam folder `/wp-content/plugins/`.
1. Hasilnya akan seperti: `./wp-content/plugins/radio-islam/*`
1. Aktifkan plugin

= Aktifasi Widget = 

1. Kini Anda bisa mengaktifkan widget RII dari "Appearance" > "Widget"
1. Drag dan drop widget "Radio Islam" ke widget area
